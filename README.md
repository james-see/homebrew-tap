# Homebrew Tap

Homebrew formulae for james-see projects.

## Formulas

### synthtribe2midi  
Converts between MIDI and Behringer SynthTribe formats (.seq/.syx). Includes CLI, TUI, and a simple REST API server for batch conversions and automation. Great for TD‑3 workflows and pattern migration. Install: ‎⁠brew install james-see/tap/synthtribe2midi⁠. ​⁠https://libraries.io/go/github.com%2Fjames-see%2Fsynthtribe2midi

### swatchify  
CLI that extracts dominant image colors via k‑means clustering, returning compact palettes for design and theming. Useful for branding, UI accents, and dataset color analysis. Install: ‎⁠brew install james-see/tap/swatchify⁠. 

## Casks  

### ynot  
Graphical YouTube video downloader packaged as a cask for easy app installs/updates. Updated to v0.5.1 in your tap’s Casks folder. Install: ‎⁠brew install --cask james-see/tap/ynot⁠. 

## Install, update, remove

- Tap: ‎⁠brew tap james-see/tap⁠

- Install (short names after tap):

 ▫ ‎⁠brew install synthtribe2midi⁠

 ▫ ‎⁠brew install swatchify⁠

 ▫ ‎⁠brew install --cask ynot⁠

- Update: ‎⁠brew update && brew upgrade⁠

- Uninstall:

 ▫ ‎⁠brew uninstall synthtribe2midi⁠

 ▫ ‎⁠brew uninstall swatchify⁠

 ▫ ‎⁠brew uninstall --cask ynot⁠

- Remove tap: ‎⁠brew untap james-see/tap⁠
