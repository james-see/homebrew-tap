# Homebrew Tap

Homebrew formulae for james-see projects.

<dia:simple-answerYour tap includes synthtribe2midi, weatherornot, gofindpi, plus swatchify and a ynot cask.></dia:simple-answer>

## Formulas

### synthtribe2midi   
Converts between MIDI and Behringer SynthTribe formats (.seq/.syx). Includes CLI, TUI, and a small REST API to batch conversions; ideal for TD‑3 workflows and pattern migration. Install: ‎⁠brew install synthtribe2midi⁠. ​⁠https://libraries.io/go/github.com%2Fjames-see%2Fsynthtribe2midi

### weatherornot  
Retro terminal weather CLI with ASCII art and charts. Pulls data from OpenWeatherMap; shows current conditions and temperature graphs. Requires a free API key. Install: ‎⁠brew install weatherornot⁠. (Serviceable for quick forecast checks and pretty terminal output.) ​⁠https://github.com/james-see/fresh-mac

### gofindpi  
Go utility to locate and identify Raspberry Pi devices on your network quickly. Useful for headless setups and DietPi boxes; reports IP and model so you can SSH in without guesswork. Install: ‎⁠brew install gofindpi⁠. ​⁠GitHub (https://github.com/james-see)

### swatchify  
CLI to extract dominant colors from images via k‑means clustering. Generates compact palettes for design, branding, and UI theming. Install: ‎⁠brew install swatchify⁠. 

## Casks

### ynot  
GUI YouTube downloader packaged as a cask for easy app install/updates; current version v0.5.1 in your Casks folder. Install: ‎⁠brew install --cask ynot⁠. 

Install, update, remove

- Tap: ‎⁠brew tap james-see/tap⁠

- Install (short names after tap):

 ▫ ‎⁠brew install synthtribe2midi⁠

 ▫ ‎⁠brew install weatherornot⁠

 ▫ ‎⁠brew install gofindpi⁠

 ▫ ‎⁠brew install swatchify⁠

 ▫ ‎⁠brew install --cask ynot⁠

- Update: ‎⁠brew update && brew upgrade⁠

- Uninstall:

 ▫ ‎⁠brew uninstall synthtribe2midi weatherornot gofindpi swatchify⁠

 ▫ ‎⁠brew uninstall --cask ynot⁠

- Remove tap: ‎⁠brew untap james-see/tap⁠

 ▫ ‎⁠brew uninstall --cask ynot⁠

- Remove tap: ‎⁠brew untap james-see/tap⁠
